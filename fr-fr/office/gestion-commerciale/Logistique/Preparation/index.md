# Vos préparations

![index-screenshotfionajoupilancom20150810160640](images/index-screenshotfionajoupilancom20150810160640.png)

Une saisie de préparation vous permet de **visualiser et de traiter les différents bons de préparations des clients**.

De cette page, vous pouvez rechercher et traiter un remboursement.

## Rechercher une saisie

Vous pouvez lancer une recherche avec les critères suivant :

1.  Les types de préparations,
2.  Le numéro du bon,
3.  Sa date de création (créée entre le... et le...),
4.  Case permettant de rechercher les non terminés uniquement.

Pour lancer la recherche, cliquez sur le bouton **GO**. (5)

## Saisie de préparation

Cette page vous donne accès à l'intégralité des saisies.

Vous pouvez avoir les informations suivantes :

1.  Le numéro du bon,
2.  Le type de préparation,
3.  Sa date de création,
4.  Sa date de préparation,
5.  Son état.

Pour traiter votre bon de préparation, cliquez sur le bouton bleu "[Ouvrir](/fr-fr/office/gestion-commerciale/Logistique/SaisiedePreparation/saisieprepaexterne.html)". (J)
